II) SETUP & DEPENDENCIES

The following dependencies must be installed for the program to run.

  • sudo apt-get install python-pip
  • pip install BeautifulSoup4
  • pip install requests
  • pip install selenium
You may need to install lxml as your parser
  • pip install
  • The folling drivers must be installed
Depending on which browser driver you wish to use

Gecko Driver for use with Firefox

For 32 bit systems

  • wget https://github.com/mozilla/geckodriver/releases/download/v0.11.1/geckodriver-v0.11.1-linux32.tar.gz
  • tar -xvzf geckodriver-v0.11.1-linux32.tar.gz
  • sudo mv geckodriver /usr/bin

For 64 bit systems

  • wget https://github.com/mozilla/geckodriver/releases/download/v0.11.1/geckodriver-v0.11.1-linux64.tar.gz
  • tar -xvzf geckodriver-v0.11.1-linux64.tar.gz
  • sudo mv geckodriver /usr/binui

PhantomJS Driver for use with PhantomJS

For 32 bit systems

  • wget https://bitbucket.org/ariya/phantomjs/downloads/phantomjs-2.1.1-linux-i686.tar.bz2
  • tar -xjzf phantomjs-2.1.1-linux-i686.tar.bz2
  • sudo mv bin/phantomjs /usr/bin

For 64 bit systems

  • wget https://bitbucket.org/ariya/phantomjs/downloads/phantomjs-2.1.1-linux-x86_64.tar.bz2
  • tar -xjzf phantomjs-2.1.1-linux-x86_64.tar.bz2
  • sudo mv bin/phantomjs /usr/bin

III) PROGRAM EXECUTION

PLEASE NOTE:

All command line arguments given to the program that contain spaces (i.e. "John Smith") must have the spaces replaced with underscores (i.e. John_Smith) for the program to execute

Running gecko_bot.py

For UNIX systems

./gecko_bot.py

Example:

./hypeswype_bot.py jackets Mohair Blue Large Mike_Smith abc@gmail 6151231234 274_Lafayette_Street 10012 New_York_City NY USA Visa 4123123412341234 01 2020 123 phantom

For Windows Systems

python hypeswype_bot.py

Example:

python hypeswype_bot.py jackets Mohair Blue Large Mike_Smith abc@gmail 6151231234 274_Lafayette_Street 10012 New_York_City NY USA Visa 4123123412341234 01 2020 123 phantom b

IV) PROFILES

If you prefer to not use command line arguments you need to fill out profile.dat in the root directory. The program will always accept the profile.dat over command line arguments.

NOTE: When filling out the profile, exclude the "". cctype is case sensitive.

NOTE: Use underscores '_' instead of spaces when filling out your profile

  • category="category"
  • keyword="keyword"
  • color="color"
  • size="size"
  • name="your first and last name as seen on credit card"
  • email="your email"
  • tel="your phone number"
  • addr="your address for shipping/billing"
  • zip="your zip code for shipping/billing"
  • city="your city for shipping/billing"
  • state="your state for shipping/billing"
  • country="your country"
  • cctype="Visa/Mastercard/American_Express"
  • cc="your credit card number"
  • expmo="the expiration month on your credit card"
  • expyr="the expiration year on your credit card"
  • cvv="the cvv number"
  • driver="the browser you wish to use (phantom or firefox)"
